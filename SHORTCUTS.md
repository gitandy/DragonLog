DragonLog - Keyboard Shortcuts
==============================

Main actions
------------

| Action         | Shortcut         |
|----------------|------------------|
| Log QSO        | Ctrl + L         |
| Edit QSO(s)    | Ctrl + E         |
| Delete QSO(s)  | Ctrl + X         |
| Watch file     | Ctrl + W         |
| Show filter    | Ctrl + F         |
| Reset filter   | Ctrl + R         |
| Show console   | Ctrl + K         |
| Check eQSL     | Ctrl + Alt + E   |
| Download eQSL  | Ctrl + Shift + E |
| Check LotW     | Ctrl + Alt + L   |
| Quit           | Ctrl + Q         |


QSO form actions
----------------

| Action         | Shortcut         |
|----------------|------------------|
| Save QSO       | Ctrl + S         |
| Search HamQTH  | Ctrl + Alt + H   |
| Search QRZCQ   | Ctrl + Alt + Z   |
| Start time now | Ctrl + Alt + N   |
